MuseMark Privacy Policy

Effective date: 2026-02-12

Data We Process

  • Bookmarks and metadata you save in MuseMark.
  • Optional AI processing input (captured page text snippet).
  • Account identity data from Supabase Auth.

Local-only Secrets

Your AI API key remains in local extension storage and is never synchronized to cloud storage by design.

Cloud Sync

If enabled, bookmark metadata and selected settings are synced to your Supabase project under your account.
